# 🔧 Araç Bakım Takip

Araç servis ve bakım atölyeleri için geliştirilmiş, **tamamen ücretsiz** ve **internet bağlantısı gerektirmeyen** mobil uygulama.

🌐 **Canlı uygulama:** https://makaraprog.github.io/arac_bakim/

---

## ✨ Özellikler

- 👤 **Müşteri Yönetimi** — Ad, soyad, telefon, adres, e-posta kaydı
- 🚗 **Çoklu Araç Desteği** — Her müşteriye birden fazla araç eklenebilir
- 🔧 **Bakım Kayıtları** — 26+ bakım kalemi (yağ, fren, filtre, lastik, elektrik...)
- 📄 **PDF Rapor** — Profesyonel bakım raporu oluşturma ve indirme
- 📱 **WhatsApp Paylaşımı** — PDF raporu direkt müşteriye gönderme
- 📊 **Dashboard** — Aylık gelir grafiği ve istatistikler
- 🔍 **Arama** — Müşteri adı, plaka veya telefon ile arama
- 📴 **Offline Çalışır** — İnternet olmadan tam fonksiyonlu çalışır
- 💾 **Veri Yedekleme** — JSON formatında dışa aktarma

---

## 📱 Kurulum

### Android
1. Chrome ile https://makaraprog.github.io/arac_bakim/ adresini açın
2. Sağ üst menü (⋮) → **"Ana ekrana ekle"**
3. Uygulama gibi açılır ✅

### iOS (iPhone/iPad)
1. Safari ile https://makaraprog.github.io/arac_bakim/ adresini açın
2. Alt menüden paylaş butonu → **"Ana Ekrana Ekle"**
3. Uygulama gibi açılır ✅

### Windows
1. Edge veya Chrome ile siteyi açın
2. Adres çubuğundaki **yükle** ikonuna tıklayın
3. Masaüstüne uygulama olarak kurulur ✅

## 🛠️ Teknolojiler

- Saf HTML5 / CSS3 / JavaScript (framework yok)
- IndexedDB (yerel veritabanı)
- Service Worker (offline destek)
- jsPDF (PDF oluşturma)
- Web Share API (WhatsApp paylaşımı)
